# Sample-Superstore Project
📊 SALES DATA ANALYSIS CASE STUDY
Tools Used: Excel | Python (Pandas, Matplotlib) | SQL (SQLite via Python)
________________________________________
📌 1. Project Overview
This project analyzes retail sales data to uncover insights into sales performance, profitability, customer behavior, and the impact of discounts.
The analysis was conducted using both Python (Pandas) and SQL within Python, allowing for cross-validation of results and demonstrating strong analytical capabilities.
________________________________________
🎯 2. Business Problem
The objective of this analysis was to:
•	Identify top-performing categories, regions, and products
•	Analyze profit and profit margins
•	Identify high-value customers
•	Evaluate the impact of discounts on profitability
•	Support data-driven business decisions
________________________________________
🗂️ 3. Dataset Description
The dataset contains transactional sales data with the following key fields:
•	Order Date
•	Category
•	Product Name
•	Customer Name
•	Region
•	Sales
•	Profit
•	Discount

📈 8. Key Insights
Category Performance
•	Technology generates the highest sales
•	Indicates strong demand and revenue contribution
Regional Performance
•	West region leads in total sales
•	Represents the strongest geographic market
Product Performance

Customer Insights
